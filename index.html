Como funciona

O servidor monta a query concatenando diretamente o input do usuário na string SQL.

Código vulnerável (PHP):
"SELECT * FROM users WHERE username='$user' AND password='$pass'"

Ao inserir caracteres especiais como ' ou --, o atacante consegue manipular a lógica da query.

Payloads prontos — clique para usar

' OR '1'='1
Torna a condição sempre verdadeira → retorna todos os usuários
admin'--
Comenta a verificação de senha → loga como admin sem senha
